$OM
Exchange Rates
Apr 2020 - July 2020
App Store
$OM is an application for reviewing exchange rates and calculating SOM (National currency of Kyrgyz Republic) into foreign currencies. Designed by techies for techies. The product is entirely non-profit and will always remain so, that is, free to use and without any advertising integrations.

Walkthrough by main points

Swipe as the best way for fast navigating

Exchange rates

Collection view for currencies
List of exchange points depends on the rolling currencies block. User can change the ordering of list according to his needs

Converter

Enter the required amount
Converter can be also an as isolated feature where user be able instantly change currency type or change to another exchange point

Settings

Customize for yourself
User be able change default currency and include or exclude exchange points

Light mode

Re-pick color mode
Application can handle both color modes. Even if user want to change color mode different from the one selected in iOS

World Nomad Games 2016
Sport Event
Apr 2016 - Jan 2017
App Store
WNG 2016 — is the only world event that brings together all sports and cultural events of the nomadic peoples of the world in one place. The app covers the most important events of ethnosport: Ordo, Kok-Boru, Wrestling, archery, Toguz Korgul and others.
App features: Purchase tickets for the Grand opening and closing of the Games, Information about the results of the games, news, photos and videos, bus schedules, Information about teams and athletes, the ability to buy ethno Souvenirs, and order taxi services.

github.com
The mobile screen has a more confined space than desktop. Have to be careful with the design layout of application interfaces, to create ergonomically right decisions. In some cases, we have multi-action buttons without a primary action. Putting all buttons to interface or hiding to a modal view can be wrong behavior for some solutions.

The collection view is the primary tool for presenting and navigating products in an App Store and other similar e-commerce platforms. Number Mask
Feauture
github.com
In some cases, during the process registration, cannot design through social networks like e-wallets, bank accounts, etc. In such cases, usually, in applications, for ease of use, developed a particular form for a phone number with a pre-selected country code and convenient number entry. But exist the case when users also should be able to sign up by e-mail. For such an example, we have designed the unique cellular number mask that appears when the user starts typing numbers.
